# Cane Easel for Oboe

## Description
The cane easel is used for marking the center of the cane and providing support when shaving the end of the cane.

![Cane Easel](/assets/images/cane-easel/cane-easel.jpeg)


## Build Guide

### Bill of Materials

#### Printed Parts
- Easel body $\times$ 1
- Easel lid $\times$ 2

#### Customization

The cane easel is fully parametric and can be customized by modifying the parameters in `cane-easel.scad`.
You can edit them to fit different instruments.
Reference parameters for oboe and English horn are given below.

| Instrument   | Easel Length | Easel Diameter |
| ------------ | ------------ | -------------- |
| Oboe         | 76mm         | 10.0mm         |
| English Horn | 93mm         | 13.7mm         |


#### Printing Tips
- Use a layer height lower than 0.12mm to make sure the screw fits well.


### Assembly
Simply screw the two easel lids onto each side of the easel body.
